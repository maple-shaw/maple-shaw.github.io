创建数据库

命名规则：

  • 可以由字母、数字、下划线、＠、＃、＄
  • 区分大小写
  • 唯一性
  • 不能使用关键字如 create select
  • 不能单独使用数字
  • 最长128位

创建表

字段名3 类型[(宽度) 约束条件]

  2. 宽度和约束条件可选
