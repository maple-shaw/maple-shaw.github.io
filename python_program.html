函数默认参数为可变类型（[ ]）

依次输出：
[10, ‘a’]
[123]
[10, ‘a’]
True
因为list1和list3调用函数extend_list时，没有传入list关键字参数，使用了函数初始化的list。
返回给list1和list3的都是初始化list的地址，所以值相同。

类的继承

依次输出：
1 1 1
1 2 1
3 2 3
因为，Child1、Child2都继承Parent类。
第一次都输出Parent的x值，为1。
第二次，Child1拥有自己的属性x=2,他的输出变为2。
第三次，Parent的属性x被赋值为3。不改变Child1自己的属性x。
